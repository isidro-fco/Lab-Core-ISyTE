CONTROL ESCOLAR

El sistema Control Escolar es una plataforma web institucional diseñada para administrar de forma centralizada la información académica y administrativa de carreras, planes de estudio, materias, docentes y alumnos. Surge como una versión mejorada del sistema existente, tras un proceso de ingeniería inversa que permitió reorganizar y formalizar sus requerimientos. Incluye módulos para gestión de usuarios, control de acceso por roles, registro y validación de datos académicos, carga de PUA’s, notificaciones y consultas públicas, todo operando en un entorno seguro, responsivo y conectado a una base de datos centralizada. Su propósito es optimizar procesos, reducir errores, mejorar la trazabilidad y fortalecer la eficiencia y transparencia académica dentro de la institución.

HORA UNIVERSITARIA

El Sistema de Gestión de Horas Universitarias es una plataforma web institucional que centraliza la planeación, registro y seguimiento de las actividades del programa, permitiendo administrar usuarios, roles, actividades, expositores, asistencias y constancias desde un entorno seguro y responsivo. Digitaliza procesos que antes eran manuales, asegurando trazabilidad, validaciones automáticas y un historial claro por alumno, mientras genera reportes y constancias oficiales para alumnos y ponentes. Su propósito es optimizar la operación del programa, reducir errores, agilizar la comprobación académica y fortalecer la transparencia y eficiencia en la gestión de Horas Universitarias dentro de la facultad.

GESTIÓN DE SALIDAS

El Sistema de Gestión de Solicitudes (SGS) es una plataforma web interna de la FIAD UABC diseñada para agilizar y controlar el proceso de solicitudes de salida de los docentes, gestionando su ciclo completo desde la creación hasta la firma final y el reporte en PDF. El sistema organiza sus funciones en tres módulos: Gestión de Usuarios, Creación y Gestión de Solicitudes, y un Historial con filtros avanzados para consultas. Incluye integración con el correo institucional para notificaciones automáticas, generación de documentos como el Oficio de Comisión y herramientas para visualizar y cargar PDFs. Dirigido a docentes y al personal administrativo con distintos niveles de permisos, el SGS opera dentro de la red UABC y en navegadores modernos, ofreciendo una interfaz intuitiva que permite firmar, rechazar, consultar y monitorear solicitudes de manera eficiente.
